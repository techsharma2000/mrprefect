
--------------
START APPLICATION
--------------

For start the application start a local instance of MongoDB, open a terminal in this folder and type:

npm install
node app.js


Go to http://localhost:3000


--------------
CONFIGURE
--------------

For NodeJS and database config edit
    properties.js

For AngularJS edit
    public/properties.js


--------------
SECURITY
--------------

Edit getUser() function in /security/security.js and get the user from your databse or LDAP
